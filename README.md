# 📚 Book Borrowing Platform

## Purpose
A modern digital library platform where users can browse books, search by title, filter by category, and manage their profile securely.

## Live URL
https://book-borrowing-platform-lac.vercel.app/

## Key Features
- User Authentication (BetterAuth)
- Email/Password & Google Login
- Protected Routes
- Search Books by Title
- Filter Books by Category
- Book Details Page
- User Profile Management
- Update Profile Information
- Responsive Design
- Custom 404 Page

## NPM Packages Used
